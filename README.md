# Cole-Jason-Fetterman-Portfolio-Website
This project is a personal portfolio site built using semantic HTML5 and custom CSS. It highlights responsive design, accessibility best practices, and layout techniques like Flexbox and CSS Grid, as well as the use of forms, tables, and animations.
